# Miguel Cotrina | Data & IA - GitHub Pages

Landing profesional premium para centralizar perfiles, credenciales, proyectos y presencia digital.

## Archivos incluidos

- `index.html`: estructura principal de la página.
- `styles.css`: estilos visuales responsive.
- `assets/miguel-cotrina.jpg`: foto profesional optimizada.
- `assets/favicon.svg`: favicon simple.
- `README.md`: instrucciones de despliegue.

## Cómo publicarlo en GitHub Pages

1. Crear un repositorio público llamado:

   `macespinoza.github.io`

2. Subir todos los archivos de esta carpeta a la raíz del repositorio.

3. En GitHub, entrar a:

   `Settings → Pages`

4. Configurar:

   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`

5. La página quedará disponible en:

   `https://macespinoza.github.io`

## Personalización rápida

Para cambiar textos, editar `index.html`.

Para ajustar colores, editar las variables dentro de `:root` en `styles.css`.

Para cambiar la foto, reemplazar:

`assets/miguel-cotrina.jpg`
